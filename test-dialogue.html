Dialogue System Test Instructions

This test page allows you to verify the dialogue system implementation:

  • Start Game: Click "Start New Adventure" to enter the game world
  • Test Dialogue: Press D key to trigger random dialogue events
  • Dialogue Controls:
    • Press SPACE or ENTER to advance dialogue
    • Click anywhere to advance dialogue
    • Click on choice buttons to select dialogue options
    • Press ESC to close dialogue
  • Other Test Keys:
    • T - Trigger random story events
    • P - Unlock power (telekinesis)
    • F - Toggle story flags
    • R - Reset story progress

Expected Behavior:

  • Dialogue box should appear at bottom of screen with speaker name
  • Text should type out character by character
  • Continue arrow should appear when text is complete
  • Choice buttons should appear for dialogue with multiple options
  • Game should pause during dialogue and resume when closed
